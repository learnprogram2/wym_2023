
# 英语学习大概记录

## 时间

去年(2022)9月, 嘉欣离开后, 我意识到自己应该学英语了, 马上有了很大的紧张感.

然后我开始备考雅思.

今年(23)1月7日第一次考, G类(7分)

上上周(3月5日)考A类(6.5, min_single_score=5.5).

虽然略有遗憾, 但也够了, 在有限中寻找自由.

## 备考内容

备考的过程不算难, 我主要是:
1. 早上练习1道听力, 临考前练习2道.
2. 中午练习1道阅读, 临考前练习1小时.
3. 晚上练习一篇作文, 有时候会漏, 大概一周3.x篇吧.
4. 晚上练习口语, 之前是对着雅思哥练习, 后面找菲教练习.
